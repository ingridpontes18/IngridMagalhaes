# Sistema de Agendamento Automatizado

Este projeto demonstra a lógica de um sistema de agendamento com controle de vagas por período.

## Funcionalidades

- Validação de datas
- Controle de vagas por período (manhã/tarde)
- Prevenção de conflitos de agendamento
- Estrutura preparada para integração com APIs e bancos de dados

## Tecnologias

- Python
- Estruturas de dados
- Lógica de automação

## Aplicações

- Clínicas
- Barbearias
- Atendimento ao público
- Sistemas de reservas

## Observação

Esta versão é uma demonstração da lógica central do sistema, sem expor integrações completas.
