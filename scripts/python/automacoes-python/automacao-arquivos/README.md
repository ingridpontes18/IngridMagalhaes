# 📁 Automação de Arquivos

Scripts voltados para **organização, padronização e conversão de arquivos** no sistema operacional.

---

## 📌 Scripts Disponíveis

### 🔹 converter_docx_pdf.py
Automatiza a conversão de documentos `.docx` para `.pdf`, mantendo padronização e reduzindo trabalho manual.

### 🔹 organizar_pastas.py
Organiza arquivos automaticamente em pastas com base em critérios como extensão ou tipo.

### 🔹 renomear_pdfs.py
Renomeia arquivos PDF de forma padronizada, facilitando controle e identificação.

---

## 🎯 Casos de Uso

- Organização de grandes volumes de arquivos
- Padronização de documentos
- Automação de rotinas administrativas

---

## 🛠️ Tecnologias

- Python
- Bibliotecas padrão do sistema
