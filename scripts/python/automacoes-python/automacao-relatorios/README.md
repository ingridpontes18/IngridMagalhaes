# 📄 Automação de Relatórios

Scripts destinados à **geração automática de relatórios visuais e documentais**.

---

## 📌 Scripts Disponíveis

### 🔹 grafico_automatico.py
Gera gráficos automaticamente a partir de dados estruturados.

### 🔹 relatorio_pdf.py
Cria relatórios em PDF de forma automatizada, mantendo layout simples e profissional.

---

## 🎯 Casos de Uso

- Relatórios gerenciais
- Documentação automática
- Visualização de dados

---

## 🛠️ Tecnologias

- Python
- Matplotlib
- ReportLab
