# 📊 Automação com Excel

Scripts focados em **manipulação, consolidação e geração de relatórios em planilhas Excel**.

---

## 📌 Scripts Disponíveis

### 🔹 consolidar_planilhas.py
Consolida múltiplas planilhas em um único arquivo, mantendo estrutura e integridade dos dados.

### 🔹 gerar_relatorio_excel.py
Gera relatórios automatizados em Excel a partir de dados estruturados.

---

## 🎯 Casos de Uso

- Consolidação de dados recorrentes
- Relatórios periódicos
- Redução de erros manuais em planilhas

---

## 🛠️ Tecnologias

- Python
- Pandas
- OpenPyXL
