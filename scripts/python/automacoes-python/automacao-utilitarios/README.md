# 🧰 Utilitários em Python

Scripts utilitários reutilizáveis para **tratamento, validação e padronização de dados**.

---

## 📌 Scripts Disponíveis

### 🔹 converter_datas.py
Converte datas entre diferentes formatos.

### 🔹 normalizar_texto.py
Padroniza textos removendo acentos, caracteres especiais e ajustando capitalização.

### 🔹 validar_cpf.py
Valida números de CPF conforme regras oficiais.

---

## 🎯 Casos de Uso

- Limpeza de dados
- Validação de entradas
- Padronização de informações

## 🛠️ Tecnologias
