# 🌐 Automação Web

Scripts para **coleta de dados e automação de tarefas na web**.

---

## 📌 Scripts Disponíveis

### 🔹 coletar_dados_site.py
Realiza scraping de dados de páginas web de forma estruturada.

### 🔹 download_automatico.py
Automatiza o download de arquivos a partir de URLs.

---

## 🎯 Casos de Uso

- Coleta de dados públicos
- Automação de downloads
- Integração com sistemas externos

---

## 🛠️ Tecnologias

- Python
- Requests
- BeautifulSoup
